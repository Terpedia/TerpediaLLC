# Terpedia, LLC Pitch Deck

This is the pitch deck for Terpedia, LLC, built with [Slidev](https://sli.dev/).

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser to the URL shown (typically `http://localhost:3030`)

## Building

To build a static version of the presentation:

```bash
npm run build
```

To export as PDF:

```bash
npm run export
```

## Managing Slides with MCP

This presentation can be managed using the Slidev MCP server. Use the MCP tools to:
- Add new slides
- Update existing slides
- Delete slides
- Read the presentation content

## Slide Types

Slidev supports various slide layouts:
- `default` - Standard slide layout
- `cover` - Cover/title slide
- `section` - Section divider
- `center` - Centered content
- `two-cols` - Two-column layout
- `image` - Image-focused layout
